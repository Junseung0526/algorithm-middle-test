# 고급 알고리즘 패러다임 (Advanced Paradigms)

## 1. 병렬 알고리즘 (Parallel Algorithm)
여러 프로세서가 동시에 작업을 수행하여 연산 속도를 높이는 방식입니다.
- **핵심**: 데이터 분할 전략과 프로세서 간 통신 최소화.
- **성능 지표**: 가속도 (Speedup), 효율성 (Efficiency).

## 2. 분산 알고리즘 (Distributed Algorithm)
여러 독립적인 시스템(노드)들이 네트워크로 연결되어 협력하는 방식입니다.
- **특징**: 중앙 집중 장치가 없으며, 각 노드는 부분적인 정보만 가짐.
- **중점 사항**: 내결함성 (Fault Tolerance), 일관성 유지.

## 3. 양자 알고리즘 (Quantum Algorithm)
양자 역학적 현상(중첩, 얽힘)을 활용하여 전통적인 컴퓨터보다 압도적인 성능을 내는 방식입니다.
- **주요 알고리즘**:
    - **쇼어 알고리즘 (Shor's Algorithm)**: 소인수 분해 ($O(\log n)$ 시간).
    - **그로버 알고리즘 (Grover's Algorithm)**: 정렬되지 않은 데이터 탐색 ($O(\sqrt{n})$ 시간).

---

## 4. 시험 포인트 및 팁
- 현대 컴퓨팅 환경에서의 확장성(Scalability) 관점에서 알고리즘의 필요성 이해.
- 양자 알고리즘이 기존 암호 시스템(RSA 등)에 미칠 수 있는 영향력 숙지.
